Web service


    Using the web service:

    To work with the users table

    Create operations: ...WS/user [POST]
    Update operations: ...WS/user/username [PUT]
    Delete operations: ...WS/user/username [DELETE]

    Special commands are needed for reading:
    To read all users: ...WS/user/all
    To read one user: ...WS/user/one/username

    To work with the kegs table

    Create operations: ...WS/keg [POST]
    Update operations: ...WS/keg/keg_id [PUT]
    Delete operations: ...WS/keg/keg_id [DELETE]

    Special commands are needed for reading:
    To read all kegs: ...WS/keg/all
    To read one keg: ...WS/keg/one/username
